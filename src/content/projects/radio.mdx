[Radio-Quran](https://github.com/MahmoudAbdullahAnani/Radio-Quran) Radio Quran is an online streaming platform that allows users to listen to live broadcasts of Quran recitations from various global radio stations. The project was built using React.js for the frontend, providing a clean, user-friendly interface for seamless listening. Users can easily navigate through a selection of stations, view station details, and enjoy continuous streaming. The backend was developed with Node.js and Express, ensuring a robust and reliable streaming experience. MongoDB was used to store station information and user preferences, and the platform supports responsive design, making it accessible across different devices.

## Key features include:

1. Live streaming of Quran recitations from various global radio stations.
2. Intuitive UI with easy station selection.
3. Responsive design for mobile and desktop users.
4. User authentication for saving favorite stations.
5. Backend API for managing station data and streaming connections.
