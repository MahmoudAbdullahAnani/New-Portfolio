[octopus-shop](https://octopus-shop.vercel.app/en) Octopus-Shop is a versatile e-commerce platform designed to offer a wide range of products, from clothing to electronics, all in one place. Developed using Next.js for the frontend, the platform provides a fast, SEO-friendly user experience, ensuring smooth navigation and product discovery. The backend was powered by Node.js, Express.js, and MongoDB, enabling a scalable and robust system for managing products, orders, and users.

![](https://raw.githubusercontent.com/MahmoudAbdullahAnani/New-Portfolio/refs/heads/mainhttps://raw.githubusercontent.com/MahmoudAbdullahAnani/New-Portfolio/refs/heads/main/public/projects/ecommerc.png)

## Key features include:

1. Product Categories: Offers a variety of products including clothing for men, women, and children, electronics, and games.
2. Seamless Checkout: Supports both online payment options like Visa and PayPal, as well as cash on delivery.
3. User Roles: Includes different user roles such as Admin for product management and regular users for browsing and purchasing.
4. Responsive Design: Works seamlessly across mobile, tablet, and desktop devices.
5. Responsive Design: Fully responsive layout optimized for mobile and desktop users.
6. Order Tracking: Users can view their order history and track their deliveries in real-time.
7. Payment Gateway Integration: Secured and smooth transaction flow with multiple payment methods.

This project showcases a well-rounded e-commerce experience, integrating modern web technologies to create a functional and user-friendly online shop.
