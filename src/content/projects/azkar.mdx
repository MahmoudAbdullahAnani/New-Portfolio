[Azkar](https://azkar-eg.vercel.app/) Azkar Muslims is a mobile and web application designed to help Muslims with daily remembrance (Azkar) through an intuitive, user-friendly platform. Built using React.js for the frontend, the app provides users with access to morning and evening prayers, as well as other essential daily supplications. The app uses PWA (Progressive Web App) technology, allowing users to access content offline after the first load, making it highly reliable even without an internet connection.

![](/public/projects/azkar.png)

## Key features include:

1. Electronic Tasbeeh (Rosary): Users can keep track of their Tasbeeh counts.
2. Daily Azkar: Morning, evening, and before sleep supplications.
3. Offline Mode: Thanks to PWA technology, users can continue using the app without internet connectivity.
4. Responsive Design: Works seamlessly across mobile, tablet, and desktop devices.
5. Reminders: Users can set reminders for specific Azkar times to stay consistent.

The app was built with a focus on simplicity and convenience, making it easy for users to incorporate daily Azkar into their routine.
